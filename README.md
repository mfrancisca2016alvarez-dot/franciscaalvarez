## 🚀 Featured Project

### 🩺 Tamizaje Kinésico Escolar — UCN Coquimbo

Computer vision system developed to support Kinesiology students during school health screening sessions at UCN.

Each year, 8th-grade students visit the university for postural evaluations. This project streamlines traditional manual assessments through computer vision and automated analysis.

### What it measures
- Shoulder asymmetry → potential scoliosis indicators
- Leg length discrepancy
- BMI based on manually entered height and weight
- Cervical projection → neck posture analysis from side-view images

### Built with
- Python
- MediaPipe
- OpenCV
- Streamlit
- ReportLab
